kototama browser WASM AOT demo

Both elements below are defined by kotoba-lang/wasm-webcomponent (KotobaWasmElement + kgraph.js) — this page no longer carries its own copy of that hosting code, it only supplies the `.wasm` payloads and a couple of `define()` calls. See ADR-2607061630.

demo.wasm (zero imports)

Byte-for-byte output of kotoba wasm emit src/demo.kotoba in kotoba-lang/kotoba (73 bytes, exports main() : i32 which returns 42) — instantiated with the browser's native WebAssembly API, no JVM, no Chicory, no wasmtime.

demo-kgraph.wasm (kgraph-* host imports)

Byte-for-byte output of kotoba wasm emit src/demo_kgraph.kotoba --policy src/demo_kgraph_policy.edn in kotoba-lang/kotoba (219 bytes; declares the kgraph_assert/kgraph_query host imports), backed by kgraph.js's browser-side port of kotoba's in-memory EAVT datom store.
