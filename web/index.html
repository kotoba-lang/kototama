kototama browser WASM AOT PoC

demo.wasm is the unmodified byte-for-byte output of kotoba wasm emit src/demo.kotoba in kotoba-lang/kotoba (73 bytes, zero imports, exports main() : i32 which returns 42). This page instantiates it with the browser's native WebAssembly API — no JVM, no Chicory, no wasmtime.

kgraph host-import PoC

demo-kgraph.wasm is the byte-for-byte output of kotoba wasm emit src/demo_kgraph.kotoba --policy src/demo_kgraph_policy.edn in kotoba-lang/kotoba (219 bytes; declares the kgraph_assert/ kgraph_query host imports). This element wires those imports to a browser-side port of kotoba's in-memory EAVT datom store (kgraph.js) instead of the JVM's kotoba.wasm-exec — the module asserts [1 :name "Aoi"] then queries {:find [?v] :where [[1 :name ?v]]}, all inside this page.
